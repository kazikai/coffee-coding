# Ground Rule
* 모이는 시간은 매주 화요일 6시 밥 먹고 2층
* 6시 30분 2층
* 모여서 1시간 공유 이후 1시간 코딩 ( 원하는 사람만  )
* 1달에 한번은 외부에 나가자
* 모임은 누가 주도할 것인지? 순서대로 돌아가면서.
